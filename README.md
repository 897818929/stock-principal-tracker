# 股票本金记录

一个完全本地运行的极简网页，用来记录银行卡和股票账户之间的转入、转出，并自动维护当前本金。

## 使用方式

直接用浏览器打开 `index.html`。

长期在手机上使用时，建议发布到 GitHub Pages，然后用手机打开固定网址并添加到主屏幕。

输入示例：

```text
银行转入股票20000元
股票转出银行5000
转入两万
转出一万二
```

## 数据说明

- 数据保存在当前浏览器的 `localStorage` 中
- 不需要登录
- 不需要服务器
- 不会自动上传到云端

每次新增转入、转出或设置初始本金后，建议立刻点“备份 JSON”。换手机、换浏览器或清理浏览器数据前，也一定要先备份。

## 手机使用流程

1. 第一次使用时，输入当前股票账户本金，点“设为初始本金”。
2. 每次银行转入股票账户，输入“转入 金额”，点“记录”。
3. 每次股票账户转回银行卡，输入“转出 金额”，点“记录”。
4. 每次记录后，点“备份 JSON”，把备份文件保存在手机文件夹里。
5. 换手机或数据丢失时，点“导入备份”，选择之前保存的 JSON 文件。

## GitHub Pages

GitHub Pages 只托管这个网页代码，不会保存你的本金记录。你的记录仍然只保存在当前手机浏览器里。

发布后，可以在 GitHub 仓库的 Settings -> Pages 中选择从 `main` 分支发布，入口文件是 `index.html`。
